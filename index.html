Portfolio

IoT Engineer

As IoT Engineer at PT. Telkom Indonesia, I studyied the use of IoT technology in industry(manufacturing, troubleshooting, and business models). I also developed a power meter data acquisition system and an electrical load monitoring system based on IoT. The system is sending electrical load data from the power meter to the IoT platform using ESP32 Long Range (LoRa) and ESP32 WiFi technology. The system is also sending electrical load data from the power meter to the IoT platform using HTTP and MQTT protocol.

Hardware Engineer

As Hardware Engineer at PT. ITS Tekno Sains, I Studied electronics industrial ecosystem, laptop manufacturing process, and business process of electronics company. I also prepared system for software and learning module injection to the tablet during manufacturing process. I designed power supply module for laptop using flyback converter topology and reverse engineering methodology. And then I analyzed laptop motherboard block diagram, chip that used in the motherboard, and wiring connection.

Research Assistant

I became Research Assistant at Virtual Power Plant research project that conducted by ITS, PT. PJB, and Kedaireka. I designed system that monitor solar power plant grid connection using SCADA, modbus TCP/IP protocol, and OPC Unified Architecture. I also integrated solar power plant with Internet of Things Server using various communication protocol. I implemented data acquisition system for industrial power meter using modbus RTU protocol. And then I implemented electrical load monitoring system using LoRaWAN and MQTT communication protocol.
